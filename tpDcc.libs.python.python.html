class tpDcc.libs.python.python.RollbackImporter¶

Bases: object

This class is used to restore the loaded modules in a certain time after the rollback is instantiated It does by storing the list of loaded Python modules when rollback is initialized and if the rollback is uninstalled, the modules loaded after rollback creation will be deleted, forcing Python to reload them when the module is next imported

uninstall()¶

Removes current loaded modules loaded after RollbackImporter was instantiated

tpDcc.libs.python.python.dict_merge(dict, merge_dict)¶

Recursive dict merge. Inspired by :meth:dict.update(), instead of updating only top-level keys, dict_merge recurses down into dicts nested to an arbitrary depth, updating keys. The merge_dct is merged into dct.

  • merge_dict – dct merged into dct
